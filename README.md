# Catálogo simples

trata-de de um simples catálogo de produtos.

## tecnologia

- golang
- postgres
- docker
- html

## para rodar

- Na raiz do projeto rodar os comandos

- subir banco, garantir que porta 5435 não esteja sendo usada

  ```bash
  docker compose up -d
  ```

- rodar aplicação

  ```bash
  go run main.go
  ```
